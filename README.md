# Street Food Night Market

Docs-first repo for a lightweight printable party game about trading resources, crafting dishes, and serving guests at a night market festival.

## Current status

Initial scaffold only.

This repo currently contains:
- design docs
- card-system notes
- sample declarative content
- workflow decisions for illustration and icon generation

## Intent

Build a small generator for printable cards and sheets, with declarative content definitions and browser/PDF export.
